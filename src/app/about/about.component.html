Directives in Angular

In Angular, directives are a type of class that adds behavior to an element in the DOM. They are a way to extend the HTML syntax of a web page and create reusable components.

There are three types of directives in Angular:

Components : These are the most common type of directive and are used to create reusable components. Components are a combination of an HTML template and a component class that controls the template.
Structural directives: These directives alter the structure of the DOM by adding or removing elements. Examples of structural directives include *ngFor and *ngIf.
Attribute directives: These directives alter the appearance or behavior of an element. Examples of attribute directives include ngStyle and ngClass.
